## ⑥難しかった点・次回トライしたいこと（又は機能）
- 次回以降やりたいことメモでもあるのですが、
- 勘定科目+金額だけでなく年月要素も追加したマトリックスでの登録、
- 勘定科目が増えた際のテーブルカラムの自動追加（ないしは削除）、
- 勘定科目”コード”や取引先”コード”追加によるバリデーションの操作性向上
- 前年同月比や前月比、対売上%などを四則演算をDB上でできないか、できそう（知らんけど）
